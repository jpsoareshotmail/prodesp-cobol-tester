BEM-VINDO ao setup do Sistema Legado COBOL da Prodesp!

Este eh um sistema de mainframe dos anos 2000 para validacao e processamento
de dados veiculares de Sao Paulo.

================================================================================
PRIMEIRA VEZ? ESCOLHA UMA OPCAO:
================================================================================

OPCAO A: "So quero analisar o codigo" (30 segundos)
────────────────────────────────────
1. Abra terminal (PowerShell ou Bash)
2. Navigate: cd c:\Projetos\outros\prodescp\codigo
3. Execute: python runner.py
4. Leia: README.md

Resultado: Relatorio completo + testes simulados


OPCAO B: "Quero compilar e rodar o codigo original" (30 minutos)
────────────────────────────────────
1. Instale GnuCOBOL:
   Baixe em: https://sourceforge.net/projects/gnucobol/files/
   Escolha: gnucobol-3.X-setup.exe (Windows)
   Execute e deixe as opcoes padrao
   Reinicie o terminal

2. Compile tudo:
   cd c:\Projetos\outros\prodescp\codigo
   bash build.sh

3. Execute:
   ./build/bin/PF-GAA-L004.exe

Resultado: Executaveis compilados em ./build/bin/


OPCAO C: "So quero ler a documentacao" (5 minutos)
────────────────────────────────────
Abra estes arquivos em ordem:
  1. README.md - Visao geral completa
  2. PLANO_EXECUCAO.md - Plano detalhado
  3. INSTALAR_GNUCOBOL.md - Se quiser compilar
