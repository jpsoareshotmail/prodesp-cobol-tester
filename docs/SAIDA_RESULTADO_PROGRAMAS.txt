FLUXO DE CARREGAMENTO:
═══════════════════════════════════════════════════════════════════

1. Usuário clica "🎯 Selecionar Programas"
   └─ Lista de 42 programas carrega

2. Para CADA programa:
   ├─ GET /api/programa/<nome>/dados → Carrega dados mockados
   ├─ POST /api/programa/<nome>/executar → Executa com dados
   ├─ Resultado é formatado em JSON
   └─ Exibido na caixa "Saída" da lista

3. Usuário vê:
   ├─ Nome do programa
   ├─ Descrição
   ├─ Objetivo
   ├─ Entrada esperada
   └─ RESULTADO REAL da execução (Saída)
