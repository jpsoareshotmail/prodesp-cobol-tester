NOVO RECURSO - SELECIONAR PROGRAMAS PARA TESTAR
================================================

O QUE FOI ADICIONADO:

✨ Botao verde: "🎯 Selecionar Programas"
  └─ Abre interface para escolher programas

📋 Interface de Seleção:
  ├─ Lista de todos 42 programas
  ├─ Organizado por categoria (GAA, GEV, GAT)
  ├─ Checkboxes para cada programa
  ├─ Botoes: "Selecionar Todos" / "Desselecionar Todos"
  └─ Botao: "Executar Testes Selecionados"

⚡ API Endpoints Novos:
  ├─ GET /api/programas
  │  └─ Retorna lista de todos os programas
  └─ POST /api/test/run-custom
     └─ Executa testes dos programas selecionados


COMO USAR:

1. Clique no botao verde "🎯 Selecionar Programas"

2. Janela abre mostrando:
   
   ┌─────────────────────────┐
   │ [✓ Todos] [✗ Nenhum]   │
   │                        │
   │ ═ GAA (16 programas)   │
   │ ☐ PF-GAA-L004          │
   │ ☐ PF-GAA-L005          │
   │ ... (14 mais)          │
   │                        │
   │ ═ GEV (26 programas)   │
   │ ☐ PF-GEV-L006-DB       │
   │ ... (25 mais)          │
   │                        │
   │ ═ GAT (2 programas)    │
   │ ☐ PF-GAT-L006-DB       │
   │ ☐ PF-GAT-T030-DB       │
   │                        │
   │ [▶ Executar] [✕ Fechar]
   └─────────────────────────┘

3. Selecione os programas que quer testar:
   └─ Clique nos checkboxes desejados
   └─ Ou use "Selecionar Todos" para rapidez

4. Clique em "▶ Executar Testes Selecionados"
   └─ Testes rodam apenas para os selecionados
   └─ Resultados em TEST_RESULTS_CUSTOM_*.json


EXEMPLOS:

Teste um programa:
  Selecionados: PF-GAA-L004
  Tempo: ~1ms
  Testes: 17

Teste uma categoria:
  Selecionados: Todos GEV (26 programas)
  Tempo: ~15ms
  Testes: 26

Teste mix personalizado:
  Selecionados: PF-GAA-L004, PF-GEV-T005-DB, PF-GAT-L006-DB
  Tempo: ~3ms
  Testes: 19
