================================================================================
FUNCIONALIDADES PRINCIPAIS
================================================================================

1. VALIDADOR DE PLACAS (PF-GAA-L004)
   Entrada: Placa veicular (até 10 caracteres)
   Saida: Codigo de validacao (0-34)

   Tipos de placa:
   - 00: Invalida
   - 11: Mercosul - Sao Paulo
   - 12: Mercosul - Outros estados
   - 21: Antiga - Sao Paulo
   - 22: Antiga - Outros estados
   - 33: 2 letras - Carros
   - 34: 2 letras - Motos

2. PROCESSADOR DE BANCO DE DADOS (PF-GAA-B100-DB)
   Manipula grandes tabelas de referencia
   Tabelas de até 619 KB
   Processamento em lote

3. EMPADRONIZACAO VEICULAR (PF-GEV-*)
   24 programas para processamento de dados
   Tabelas de 5 KB até 335 KB

COMO USAR AGORA
================================================================================

OPCAO 1: SEM INSTALAR NADA (Simulador)
----
1. Abra terminal na pasta do projeto:
   cd c:\Projetos\outros\prodescp\codigo

2. Execute:
   python runner.py

3. Resultado: Analise completa + testes simulados de validacao de placas

Tempo: ~10 segundos
Requisito: Python (já instalado)


OPCAO 2: COMPILAR E RODAR (Recomendado)
----
1. Instale GnuCOBOL:
   - Windows: Baixe em https://sourceforge.net/projects/gnucobol/files/
   - Execute instalador e marque "Add to PATH"
   - Reinicie terminal

2. Compile tudo:
   bash build.sh

3. Execute programa piloto:
   ./build/bin/PF-GAA-L004.exe

Tempo: ~30 minutos (inclui instalação)
Requisito: GnuCOBOL (gratuito, open-source)


OPCAO 3: COMPILACAO MANUAL
----
cd "PGM POC cob original"
cobc -x PF-GAA-L004.C74 -o validador.exe
./validador.exe

Tempo: ~2 minutos
Requisito: GnuCOBOL instalado
