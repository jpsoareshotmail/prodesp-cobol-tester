INTERFACE DE SELEÇÃO:
═══════════════════════════════════════════════════════════════════

Ao clicar em "🎯 Selecionar Programas":

1. Janela modal abre
2. Mostra 42 programas organizados:
   - GAA (16 programas)
   - GEV (26 programas)
   - GAT (2 programas)
3. Cada programa tem checkbox
4. Botoes auxiliares:
   - "Selecionar Todos"
   - "Desselecionar Todos"
5. Botao para executar:
   - "Executar Testes Selecionados"
6. Botao para fechar

ARQUIVO DE RESULTADO:
═══════════════════════════════════════════════════════════════════

Antes:
  TEST_RESULTS_*.json          (Basicos)
  TEST_RESULTS_EXPANDED_*.json (Expandidos)

Depois:
  TEST_RESULTS_*.json          (Basicos)
  TEST_RESULTS_EXPANDED_*.json (Expandidos)
  TEST_RESULTS_CUSTOM_*.json   (Customizados) ← NOVO!

Cada arquivo contém APENAS os testes selecionados


API ENDPOINTS NOVOS:
═══════════════════════════════════════════════════════════════════

GET /api/programas
  └─ Retorna lista de todos 42 programas
  └─ Organizado por categoria

POST /api/test/run-custom
  └─ Inicia testes customizados
  └─ Body: {"programas": ["PF-GAA-L004", "PF-GEV-T005-DB"]}
